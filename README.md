## 📌 Projetos em destaque

### CME Manager System

Sistema web para gestão e rastreabilidade de materiais em uma Central de Material e Esterilização.

**Principais recursos:**

- Controle de materiais, lotes e ciclos de esterilização
- Rastreamento por QR Code e código de barras
- Relatórios em PDF e CSV
- Controle de usuários e permissões
- Integração com Hyperledger Fabric

**Tecnologias:** React, TypeScript, Node.js, Express, Prisma, PostgreSQL, JWT e Tailwind CSS.

---

### is.sues

Aplicação para gestão de empresas, departamentos, usuários e chamados internos.

**Principais recursos:**

- Autenticação de usuários
- Gestão de empresas e departamentos
- Controle de chamados
- Upload de imagem de perfil
- Organização por cargos e permissões

**Tecnologias:** React Native, TypeScript, Node.js, Express, Context API e Cloudinary.

---

### Brasil Play Games

Projeto de desenvolvimento e manutenção de sistemas para servidor SA-MP RPG, envolvendo programação de gamemode, sistemas administrativos, eventos, organizações e integrações.

**Principais recursos:**

- Sistemas de organizações
- Eventos automáticos
- Rankings e logs
- Sistemas administrativos
- Integrações com Discord
- Automações e sistemas de recompensas

**Tecnologias:** Pawn, MySQL, Node.js e Discord.js.
